# 行为准则

## 社区标准

我们致力于为所有参与者提供尊重、专业、可协作的交流环境。

鼓励的行为：

- 交流中保持尊重和建设性
- 反馈聚焦代码和结果，而非个人
- 遇到分歧时先澄清上下文再升级讨论
- 提出技术意见时给出必要事实和依据

不可接受的行为：

- 骚扰、人身攻击、歧视性言论
- 挑衅、辱骂或持续性干扰讨论
- 未经许可公开他人隐私信息

## 执行方式

维护者负责解释和执行本准则。对于违反规则的评论、提交、Issue 或 PR，维护者可进行编辑、隐藏或拒绝处理。

## 举报方式

若你遭遇或目击不当行为，请通过仓库 Issue 或私下联系维护者反馈。所有举报都会被认真处理。
