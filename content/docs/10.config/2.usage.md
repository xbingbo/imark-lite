## 添加依赖
```xml
<dependency>
    <groupId>com.alibaba.cloud</groupId>
    <artifactId>spring-cloud-starter-alibaba-nacos-config</artifactId>
</dependency>
```

## 配置文件

application.yml (或 application.properties)

```yaml
spring:
  application:
    name: example-app

  cloud:
    nacos:
      config:
        server-addr: 127.0.0.1:8848
        file-extension: yaml
        namespace: public
        group: DEFAULT_GROUP
        refresh-enabled: true
  config:
    import:
      - nacos:example-app.yaml
```

## 配置定义

```java
@Data
@Component
@RefreshScope # 实现配置自动更新
@ConfigurationProperties(prefix = "app")
public class AppConfig {
    private int timeout;
    private Feature feature;

    @Data
    public static class Feature {
        private boolean enableLogin;
    }
}
```

## 创建配置

在Nacos控制台创建配置：
- Data ID: example-app.yaml
- Group: DEFAULT_GROUP
- Namespace: public
  配置内容：

```yaml
app:
  # 超时时间
  timeout: 3000
  feature:
    enableLogin: true
```
