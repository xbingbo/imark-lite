解压到`/opt/nacos`后目录结构如下：
```yaml
nacos/
├── bin/
├── conf/
├── logs/
└── target/
```

### 修改配置

修改`conf/application.properties`文件，可以用注释里的配置值

```yaml
nacos.core.auth.server.identity.key=VGhpc0lzTXlDdXN0b21TZWNyZXRLZXkwMTIzNDU2Nzg=
nacos.core.auth.server.identity.value=VGhpc0lzTXlDdXN0b21TZWNyZXRLZXkwMTIzNDU2Nzg=

### worked when nacos.core.auth.system.type=nacos or nacos.core.auth.console.enabled=true
### The token expiration in seconds:
nacos.core.auth.plugin.nacos.token.cache.enable=false
nacos.core.auth.plugin.nacos.token.expire.seconds=18000
### The default token (Base64 string):
#nacos.core.auth.plugin.nacos.token.secret.key=VGhpc0lzTXlDdXN0b21TZWNyZXRLZXkwMTIzNDU2Nzg=
nacos.core.auth.plugin.nacos.token.secret.key=VGhpc0lzTXlDdXN0b21TZWNyZXRLZXkwMTIzNDU2Nzg=
```
### 启动Nacos（默认使用内嵌数据库）

进入`bin/`目录，根据系统执行：

```shell
sh startup.sh -m standalone
```

### 创建 systemd 服务

```yaml
# /etc/systemd/system/nacos.service
[Unit]
Description=Nacos Service
After=network.target

[Service]
Type=forking
Environment=JAVA_HOME=/usr/lib/jvm/java-17-openjdk-17.0.11.9-1.oe2203sp4.x86_64
ExecStart=/bin/bash /opt/nacos/bin/startup.sh -m standalone
ExecStop=/bin/bash /opt/nacos/bin/shutdown.sh
Restart=on-failure
RestartSec=10
LimitNOFILE=65535
User=root
WorkingDirectory=/opt/nacos

[Install]
WantedBy=multi-user.target
```

### 启动服务

```shell
systemctl daemon-reload
# 启动服务
systemctl start nacos

# 查看状态
systemctl status nacos

# 设置开机自启
systemctl enable nacos

# 停止服务
systemctl stop nacos

# 重启服务
systemctl restart nacos

```
