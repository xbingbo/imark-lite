#### 第一步：写 Markdown

```md
# 这是一篇公众号示例文章

> 一段引言，快速说明这篇文章的核心价值。

![](https://images.unsplash.com/photo-1502082553048-f009c37129b9?auto=format&fit=crop&w=1200&q=80)

## 主要观点

- 用 Markdown 写作更专注
- 复制粘贴即可发布
- 公众号显示样式稳定

## 使用说明
1. 在左侧编辑区输入 Markdown 内容
2. 右侧实时显示预览效果
3. 点击「复制」，粘贴到公众号编辑器发布

> 💡 小提示：建议使用短段落与清晰小标题，提升公众号阅读体验。

---

**开始你的创作之旅吧！** ✨
```

#### 第二步：复制排版后的内容

点击「复制」，你得到的是公众号可识别的富文本

#### 第三步：粘贴到公众号编辑器

- 在公众号后台新建图文
- 粘贴到正文区域即可保持排版
- 检查无误后发布

::
