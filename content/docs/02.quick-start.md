#### 第一步：写 Markdown

```markdown
# 欢迎使用微信公众号编辑器

请在*这里*输出**markdown**来尝试~创作~！！！

![](https://images.unsplash.com/photo-1502082553048-f009c37129b9?auto=format&fit=crop&w=1200&q=80)

| Key | Type      | Description |
| --- | --------- | ----------- |
| 1   | Wonderful | Table       |
| 2   | Wonderful | Data        |
| 3   | Wonderful | Website     |


## 主要功能
- 实时预览
- 移动端适配
- 多种主题样式


## 使用说明
1. 在左侧编辑区输入 Markdown 格式的文章
2. 右侧会实时显示渲染后的效果
3. 点击底部的发布按钮即可推送到微信公众号


> 💡 小提示：使用工具栏可以快速插入常用的 Markdown 语法

---

**开始你的创作之旅吧！** ✨
```

#### 第二步：复制排版后的内容

点击「复制」，你得到的是公众号可识别的富文本

#### 第三步：粘贴到公众号编辑器

- 列表
- 标题
- 引用
- 表格
- 图片

完成发布。

::
