## 欢迎使用 imark-lite

**imark-lite** 是一个轻量级 Markdown 排版工具，为 **公众号创作者和技术写作者**设计。  
无需复杂配置，就能快速生成可直接发布的漂亮文章。

立即访问在线体验：[https://ebingbo.github.io/imark-lite/](https://ebingbo.github.io/imark-lite/)

---

## 快速开始

想在几分钟内写出第一篇公众号文章？  
请查看 [快速上手指南](/docs/quick-start) ，3 步完成文章创作与发布。

---

## 功能亮点

- **Markdown → 富文本**：文章内容可直接粘贴到公众号编辑器
- **多风格排版**：简约白 / 护眼黑 
- **实时预览**：边写边看，文章效果即时呈现
- **轻量高效**：前端 Vue + Nuxt UI，无需额外配置




> 💡 提示：在 Docs 页面中，所有示例和截图都可以直接复制到 imark-lite 在线工具中体验。
